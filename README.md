# 🔐 Gerador de Senhas em Python

Este projeto gera senhas aleatórias com base no tamanho escolhido pelo usuário.

## 🚀 Funcionalidades
- Escolha do tamanho da senha
- Geração de senha com letras, números e símbolos
- Execução simples via terminal

## 💻 Tecnologias
- Python
